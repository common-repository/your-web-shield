== Description ==
Your Web Shield blocks high-risk IPs and limits request rates, providing enhanced security for your site.

== Installation ==
1. Upload `yourwebshield` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the "Your Web Shield" menu to configure settings like risk threshold and rate limiting.

== Frequently Asked Questions ==
= What is a risk threshold? =
The risk threshold determines the level of risk at which an IP will be blocked. You can configure this from the plugin settings.

== External Services ==
This plugin relies on the external service "Your Web Shield API" to check IP addresses for risk scoring.
